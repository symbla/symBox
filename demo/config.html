Einstellungen

Aussehen

CSS-Klasse

Beschreibung

.no-border

Entfernt den Rahmen um dieses Element (border: none).

.no-shadow

Entfernt den Schatten dieses Elements (box-shadow & text-shadow).

Positionierung

CSS-Klasse

Beschreibung

.fullscreen

Ändert das "position-Property" dieses Elements zu "absolute" und setzt die Abstände zum oberen, rechten, unteren und linken Rand auf 0.
Das Element besitzt nun die gesamte Größe des Bildschirms.

.absolute

Ändert das "position-Property" zu "absolute".

.relative

Ändert das "position-Property" zu "relative".

.fixed

Ändert das "position-Property" zu "fixed".

Docking

CSS-Klasse

Beschreibung

.dock-top

Dockt ein Element am oberen Bildschirmrand an.
(Nur in verbindung mit Klasse .absolute oder .fixed)

.dock-right

Dockt ein Element am rechten Bildschirmrand an.
(Nur in verbindung mit Klasse .absolute oder .fixed)

.dock-left

Dockt ein Element am unteren Bildschirmrand an.
(Nur in verbindung mit Klasse .absolute oder .fixed)

.dock-left

Dockt ein Element am linken Bildschirmrand an.
(Nur in verbindung mit Klasse .absolute oder .fixed)

Textausrichtung

CSS-Klasse

Beschreibung

.text-center

Text in diesem Element wird nach links ausgerichtet.

.text-justify

Text in diesem Element wird im Blocksatz ausgerichtet.

.text-right

Text in diesem Element wird nach rechts ausgerichtet.

Innere Abstände (Padding)

CSS-Klasse

Beschreibung

.no-pad

Die Padding-Eigenschaft für dieses Element wird entfernt.

.no-pad-top

Die Padding-Top-Eigenschaft für dieses Element wird entfernt.

.no-pad-right

Die Padding-Right-Eigenschaft für dieses Element wird entfernt.

.no-pad-bottom

Die Padding-Bottom-Eigenschaft für dieses Element wird entfernt.

.no-pad-left

Die Padding-Left-Eigenschaft für dieses Element wird entfernt.

.pad-top

Der Abstand am oberen inneren Rand dieses Elements wird auf 10 Pixel gesetzt.

.text-center

Der Abstand am rechten inneren Rand dieses Elements wird auf 10 Pixel gesetzt.

.text-justify

Der Abstand am unteren inneren Rand dieses Elements wird auf 10 Pixel gesetzt.

.text-right

Der Abstand am linken inneren Rand dieses Elements wird auf 10 Pixel gesetzt.

Äußere Abstände (Margin)

Beschreibung

Die Einstellungen für die äußeren Abstände sind identisch mit den Klassen für innere Abstände. Es muss lediglich das pad durch mar ersetzt werden.
